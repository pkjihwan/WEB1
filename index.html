흡연의 안 좋은 점

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

뇌졸증
뇌혈류 이상으로 생기는 뇌졸중은 갑작스런 뇌 기능에 장애를 일으키는 증상입니다. 우리 몸에서 아주 중요한 역할을 하는 뇌는 문제가 발생되면 아주 치명적인 결과를 가져 옵니다. 흡연이 혈관에 손상을 주며 혈액순환에 방해를 해서 매우 위험에 상황에 처할 수 있는 질환입니다.

암
또다른 흡연 안좋은점을 꼽자면 오래동안 담배를 할수록 암이 발생되는 확률은 높아집니다. 폐암과 다양한 암을 유발하기도 하는데 담배에 7000가지 이상 독성화학 물질이 내포되어 있어서 식도암, 위함, 직장암, 폐암 등과 같은 다양한 암을 유발합니다.

심혈관 질환
담배를 오래 피우는 사람들은 흔히 발생하는 질환이 폐암이라 생각을 하는 사람들이 많은데 이보다 심혈관 질환으로 사망하는 흡연자가 더 많습니다. 심혈관 질환은 사망 원인의 2위를 기록할 정도로 우리나라에서 많이 발생하는 질환이며 심장, 동맥 증에 어떤 형태로 손상되고 질환이 발생된다 는 것을 뜻합니다.
